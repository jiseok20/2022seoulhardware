# 2022-Seoul-Hardware-Hackathon

### game
디스플레이에 보이는 프로그램의 라즈베리파이에서 사용하는 코드입니다.  
img : 화면 파일  
mp3 : 음성 파일  

### client.py
라즈베리카메라에서 영상을 찍고 server에 영상을 전송하는 라즈베리파이에서 사용하는 코드입니다.  

### server
라즈베리파이, stm32와 통신하는 코드입니다.  
pycham을 사용하였습니다.  

### main.zip
stm32에서 사용하는 코드입니다.  
STM32 cubeIDE 환경에서 제작하였습니다.
